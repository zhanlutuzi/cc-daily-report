一个普通工作日

闭环，从头到尾。

一天里四个时刻。两个你主动触发，两个自动跑。四个全部写到同一个地方。

早上~09:00

站会回顾

/morning-review

读过去 36h 的摘要卡片，提炼一份回顾，主动问你 2-3 个关于今天的具体问题。就这样 — 后面的对话从这里继续。

每次 Claude Code 压缩会话时，hook 静默把压缩前的摘要存到 ~/.claude/summaries/。你完全无感，但东西都在。

开始使用

三步。

或者直接 clone 仓库，让 Claude Code 帮你装。README 里有现成的 prompt。

1

拷贝文件

在仓库根目录执行：

# commands 和 hooks 拷到 ~/.claude/
cp commands/*.md ~/.claude/commands/
cp hooks/*.sh   ~/.claude/hooks/
chmod +x ~/.claude/hooks/*.sh
mkdir -p ~/.claude/summaries
2

注册 PostCompact hook

合并到 ~/.claude/settings.json：

{
  "hooks": {
    "PostCompact": [{
      "matcher": "*",
      "hooks": [{
        "type": "command",
        "command": "bash \"$HOME/.claude/hooks/compact-summary-card.sh\"",
        "timeout": 15
      }]
    }]
  }
}
3

注册 crontab（可选）

每天 18:00 — 改成你想要的时间：

(crontab -l 2>/dev/null; \
  echo "0 18 * * * /bin/bash $HOME/.claude/hooks/daily-summary-cron.sh >> $HOME/.claude/summaries/cron.log 2>&1") \
  | crontab -

⚠ cron 环境的 PATH 极简 — 编辑 daily-summary-cron.sh 第 14 行，把 claude 和 jq 所在路径加进去。
